# 风格画像模板

用于快速抽取“用户文档风格”，保证生成结果可直接用。

## 1. 结构偏好

- 是否采用完整PRD结构（1~6章）：
- 功能清单是否固定在4.2：
- 是否单列“需求范围说明”：
- 5章是否“每页一图一表”：
- 是否禁用总结章节：

## 2. 表格偏好

- 功能清单表头：
- 功能详设表头（3列/4列）：
- 是否允许跨页面共用一张表：
- 列宽比例（如20/40/40）：

## 3. 元素表达偏好

- 基础句式（默认）：`字段名，字段类型，字段要求`
- 列表模块固定骨架（信息汇总/主按钮/列表）：
- 是否要求“展示范围 + 展示顺序 + 字段说明 + 字段交互”：
- 字段顺序是否强制“从上到下、从左到右”：

## 4. 逻辑表达偏好

- 重点覆盖：入参校验/状态流转/计算口径/异常处理
- 是否需要复杂字段深写（阈值、并列、边界）：
- 禁止的技术化描述：

## 5. 序号与排版偏好

- 一级序号：`1.2.3`
- 二级序号：`a.b.c`
- 同级要点是否允许空行：
- 信息架构是否使用代码块：

## 6. 术语偏好

- 关键术语：
- 禁用术语：
- 页面命名是否保留英文路径：

## 7. 交付与更新偏好

- 默认交付目标（飞书/本地）：
- 默认更新策略（局部更新/整文重写）：
- 流程图形式（飞书组件/图片/文本兜底）：

## 8. 常见禁用表达

- 待确认/待定/可选/视情况
- 研发自行决定
- 不可验收的泛化描述
