## 规则说明

1. 功能清单只写模块级能力，不写页面内原子操作。
2. 二级功能名称优先对应页面、页签或独立业务模块。
3. 上架、下架、删除、导出、下载、设为默认等动作，写入 5 章功能详细说明，不在 4.2 单独占一行。
