# 元素与逻辑示例

## 元素写法示例

1. 今日直播，列表区。
   a. 展示范围：当天所有直播课。
   b. 展示顺序：按照直播开始时间顺序展示。
   c. 课程卡片。
      i. 直播名称，文本，不折行。
      ii. 课程名称，文本，不折行。
      iii. 直播时间，文本，格式为“日期标签 + 开始时间 - 结束时间”。
      iv. 应到课人数，数字，只展示整数。

## 逻辑写法示例

1. 错误集中。
   a. 统计口径：仅统计已作答人数，未作答不进入错误分布。
   b. 判定规则：错误人数为 0 时不展示“错误集中”。
   c. 集中规则：错误选项中人数最多的选项人数 >= 2 且占错误人数比例 >= 50% 时，展示“错误集中在 X”。
   d. 并列规则：若两个错误选项并列第一，则展示“错误集中在 A/C”。
